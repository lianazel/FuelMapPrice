À propos

Une app utile, ouverte et sans pub.

FuelMapPrice est une application web gratuite qui s'appuie exclusivement sur les données officielles publiées par le gouvernement français sur data.gouv.fr.

Les prix sont rafraîchis chaque heure via une GitHub Action qui télécharge le flux officiel, l'agrège et le publie au format JSON. L'application fonctionne entièrement côté navigateur : aucun serveur, aucune collecte de données personnelles.

Stack technique

  • Front : HTML + Tailwind CSS + Alpine.js (zéro build)
  • Carte : Leaflet.js + tuiles OpenStreetMap
  • Graphiques : Chart.js
  • Géocodage ville : Nominatim (OpenStreetMap)
  • Données prix : data.gouv.fr / donnees.roulez-eco.fr
  • Hébergement : GitHub Pages

Code source et issues sur GitHub. Contributions bienvenues !
